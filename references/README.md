## References Folder

**Reference 폴더 내의 파일들을 참고할 때 필독하세요!**

- 이 폴더 내부에 있는 파일들은 다른 프로젝트(특히 SKURI React Native 앱)의 소스코드 파일들을 복제한 폴더입니다.
- 주로 이 SKURI Spring 프로젝트의 API들을 활용할 클라이언트(SKURI RN 앱, SKURI ADMIN 페이지 프로젝트 등)들의 파일들을 레퍼런스로 둡니다.

### 파일 설명
1. ```references/index.ts``` : Firebase SKURI Firebase Cloud Functions에 배포된 함수 파일.
   현재 이 함수들이 Cloud Functions에서 발동되고 있으며, 이 함수들을 점진적으로 이 SKURI Spring 프로젝트로 옮기는게 목표.
   SKURI Backend Spring 프로젝트를 구현할 때 레퍼런스로 참고하도록 한다.
